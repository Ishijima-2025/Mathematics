・ k = ω

\( \Large \displaystyle F(s) =\frac{ s \ f(0) + f'(0) + 2k \ f(0)}{s^2 + 2ks+ \omega^2 } \)

ですが，k=ω，の場合には，

\( \Large \displaystyle F(s) =\frac{ s \ f(0) + f'(0) + 2k \ f(0)}{(s+k)^2 } \)

となり，さらに，

\( \Large \displaystyle F(s) =\frac{ s \ f(0) + f'(0) + 2k \ f(0)}{(s+k)^2 }
= \frac{ (s+k) \ f(0) + f'(0) + k \ f(0)}{(s+k)^2} \)

とすると，

\( \Large \displaystyle F(s) = \frac{ f(0) }{s+k} + \frac{ f'(0) + k \ f(0)}{(s+k)^2} \)

と分けることができます．
