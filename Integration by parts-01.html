瞬間部分積分

 

部分積分，結構面倒ですね．特に三角関数の場合には，二回部分積分を行わなくてはならない．

しかし！最近，瞬間部分積分，なるものを見つけました．すでに当たり前になっている手法かもしれませんが．こちら，を参考にしました

その条件は，

\( \Large \displaystyle \int f(x) g(x) dx \)

において，

　f(x)を何回か微分したら０になる．
　g(x)は積分可能

な場合に成り立つようです．その手順とは，

　１．f(x)を微分して，定数となるまで続ける
　２．g(x)を積分していく
　３．"+,-,+,-"と符号をつける
　４．１．２．３の積を作る
　５．定数を加える

です．

 

・簡単な例1-1

\( \Large \displaystyle \int x \ e^x dx \)

を考えていきます．真っ当に解くと，

\( \Large \displaystyle (fg)' = f'g + fg' \)

\( \Large \displaystyle f'g = (fg)' - fg' \)

\( \Large \displaystyle \int f'g \ dx= \int (fg)' \ dx- \int fg' \ dx = fg - \int fg' \ dx\)

となるので，

\( \Large \displaystyle f' = e^x, \ g =x \)

とすれば，

\( \Large \displaystyle f = e^x, \ g' =1 \)

となるので，

\( \Large \displaystyle \int x \ e^x dx = e^x x - \int e^x \ dx = x e^x - e^x + C \)

です．瞬間部分積分を使えば，

  微分回数   積分回数  
+ 0 x 1 ex
- 1 1 2 ex

\( \Large \displaystyle \int x \ e^x dx = x e^x - e^x + C \)

・簡単な例1-2

\( \Large \displaystyle \int x \ e^{ax} dx \)

  微分回数   積分回数  
+ 0 x 1 \( \Large\frac{1}{a} e^{ax} \)
- 1 1 2 \( \Large\frac{1}{a^2} e^{ax} \)

\( \Large \displaystyle \int x \ e^{ax} dx = \frac{1}{a} x \ e^{ax} - \frac{1}{a^2} e^{ax} + C\)

 

・簡単な例2-1

\( \Large \displaystyle \int x^2 \ e^x dx \)

  微分回数   積分回数  
+ 0 x2 1 ex
- 1 2x 2 ex
+ 2 2 3 ex

\( \Large \displaystyle \int x^2 \ e^x dx = x^2 e^x -2x e^x + 2 e^x + C \)

 

・簡単な例2-2

\( \Large \displaystyle \int x^2 \ e^{ax} dx \)

  微分回数   積分回数  
+ 0 x2 1 \( \Large\frac{1}{a} e^{ax} \)
- 1 2x 2 \( \Large\frac{1}{a^2} e^{ax} \)
+ 2 2 3 \( \Large\frac{1}{a^3} e^{ax} \)

\( \Large \displaystyle \int x^2 \ e^x dx = \frac{1}{a} x^2 e^x -\frac{2}{a^2}x e^x + \frac{2}{a^3} e^x + C \)

 

 

・簡単な例3-1_三角関数を含む場合

\( \Large \displaystyle \int x \ sin \ x \ dx \)

  微分回数   積分回数  
+ 0 x 1 cos x
- 1 1 2 -sin x

\( \Large \displaystyle \int x \ sin \ x \ dx = x \ cos \ x + sin \ x + C\)

 

・簡単な例3-2_三角関数を含む場合

\( \Large \displaystyle \int x \ sin \ ax \ dx \)

  微分回数   積分回数  
+ 0 x 1 \( \Large -\frac{1}{a} cos \ ax \)
- 1 1 2 \( \Large \frac{1}{a^2} sin \ ax \)

\( \Large \displaystyle \int x \ sin \ ax \ dx = - \frac{1}{a} x \ cos \ x - \frac{1}{a^2}sin \ x + C\)

 

・簡単な例3-3_三角関数を含む場合

\( \Large \displaystyle \int x \ cos \ x \ dx \)

  微分回数   積分回数  
+ 0 x 1 sin x
- 1 1 2 -cos x

\( \Large \displaystyle \int x \ sin \ x \ dx = x \ sin \ x + cos \ x + C\)

 

・簡単な例3-4_三角関数を含む場合

\( \Large \displaystyle \int x \ cos \ ax \ dx \)

  微分回数   積分回数  
+ 0 x 1 \( \Large \frac{1}{a} sin \ ax \)
- 1 1 2 \( \Large -\frac{1}{a^2} cos \ ax \)

\( \Large \displaystyle \int x \ sin \ ax \ dx = \frac{1}{a} x \ sin \ x + \frac{1}{a^2}cos \ x + C\)

 

・簡単な例3-5_三角関数を含む場合

\( \Large \displaystyle \int x^2 \ cos \ x \ dx \)

  微分回数   積分回数  
+ 0 x2 1 sin x
- 1 2x 2 -cos x
+ 2 2 3 -sin x

\( \Large \displaystyle \int x^2 \ cos \ x \ dx = x^2 sin \ x-2x \ cos \ x - 2 sin \ x + C =(x^2 - 2) sin \ x -2x \ cos \ x + C\)
